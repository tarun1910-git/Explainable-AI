About Endoscopy AI System

The Endoscopy AI System is designed to assist in the detection of gastrointestinal diseases using advanced Artificial Intelligence techniques.

The system analyzes endoscopy images and classifies them into different medical categories using deep learning models.

Disease Categories
  • Bleeding
  • Ulcer
  • Polyp
  • Normal
Technologies Used
  • Deep Learning Models (MobileNet / ResNet)
  • TensorFlow / Keras
  • Django Web Framework
Explainable AI

The system incorporates Explainable AI techniques such as Grad-CAM and LIME to highlight important regions in the image. This helps in understanding how the model makes decisions.

Objective

The main objective of this project is to support early detection of gastrointestinal diseases and assist medical professionals by providing reliable and interpretable predictions.
